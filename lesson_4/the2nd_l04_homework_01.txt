# the2nd_l04_homework_01

Даны файлы в папке movies: index.html, movies.html, ratings.html, contacts.html

Перейти в папку LESSON_04 и создать в ней папку the2nd_l04_homework_01. Перейти в папку the2nd_l04_homework_01. Скопируйте в нее папку movies.

A. Cтраница index.html

1) замените изображения заголовка на странице, используя внешнюю ссылку (https://url.address)
2) замените 5 режиссеров на своих любимых
3) используйте изображения для них найдите на сайте kinopoisk.ru (размеры фото 280x420); укажите атрибут height для тэга <img> равный 105px
4) для каждого режиссера добавьте ссылку на сайт wikipedia, которая будет открываться в новом окне
5) В разделе мои любимые актеры добавьте 5 своих любимых актеров / актрис (по аналогии с режиссерами: 
 Изоражения, ссылки на wikipedia)
P.S. Все фотографии для режиссеров и актеров / актрис сохранить на диске локально
6) Добавьте на страницу тэги <div>, которые должны объединить блоки:
	- Изображение заголовка
	- История кинематографа
	- Любимые режиссеры
	- Любимые актеры (топ-5)
7) Добавьте в раздел любимые режиссеры тэг <span> и поменяйте цвет для самого любимого режиссера
8) Добавьте в раздел любимые актеры тэг <span> и поменяйте цвет для самого любимого актера / актрисы

B. Страница movies.html

1) замените изображения заголовка на странице, используя внешнюю ссылку (https://url.address)
2) заполните страницу информацией о 5 ваших любимых фильмов, каждый из которых должен содержать:
	- название фильма (<h1>-<h6>)
	- изображение к фильму (<img>)
	- описание фильма (<p>)
	- список актеров (<ul>)
	- режиссер(-ы), продюсер(-ы) (<ul>)
	- каждый фильм поместить в отдельный блок (<div>)
	- внешние ссылки на kinopoisk или wikipedia (<a>)

C. Страница ratings.html

1) замените изображения заголовка на странице, используя внешнюю ссылку (https://url.address)
2) добавьте на страницу таблицу, которая должна состоять из 5 фильмов и рейтингов к ним: kinopoisk и IMDB (информация на сайте kinopoisk). Формат таблицы: Название, Kinopoisk, IMDB, источник
P.S. Например: "В погоне за счастьем", 8.2, 8.0, https://www.kinopoisk.ru/film/104938/

D. Страница contacts.html

1) замените изображения заголовка на странице, используя внешнюю ссылку (https://url.address)
2) Добавьте на страницу форму для связи:
	- Напишите мне (<h1>-<h6>)
	- Форма (<form>)
	- Поля формы:
		имя: input --> name="first_name", type="text", id="first_name"
		email: input --> name="email", type="email", id="email"
		сообщение: textarea --> name="message", id="message"
		кнопка: button --> type="submit", Отправить сообщение
